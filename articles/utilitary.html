Les classes d’Utilitary

Class ConvertTo

La classe ConvertTo est l’une des classes essentielles pour le traitement de l’image notamment pour la lecture du header des fichiers bitmap :

Elle contient 2 régions

  • FromLittleEndian contenant les méthodes permettant la conversion d’un tableau de byte en format little endian à un type désiré
    • UInt qui prends en paramètre un tableau debyte et retournes un uint
    • Int qui prends en paramètre un tableau debyte et retournes un int
    • UShort qui prends en paramètre un tableau debyte et retournes un ushort
  • ToLittleEndian contenant les méthodes permettant de convertir différents types de valeurs en un tableau de byte dans le format Little Endian

Classe Convolution

Cette classe contient toutes les matrices de convolution des différentes transformations.

Ainsi dans la classe convolution on peut trouver les deux régions suivantes :

  • Dictionnary, contenant les différents kernel (ou noyau) de chaque transformation.

Ainsi dans Dictionnary, on trouve un dictionnaire qui associe une Transformation (de l’énumération des transformations) à une matrice de float. Ainsi nous avons le kernel de :

  • Détection de contours
  • Renforcement des bords
  • Flou
  • Repoussage
  • Extensions methods for Image, contenant des méthodes annexes pour appliquer les kernel donnés précédemment sur nos images bitmap. Ainsi, on a les méthodes suivantes :
    • ApplyKernel qui prend en entrée l’image sur laquelle il faut appliquer la transformation et la matrice de float ici appelée kernel.
    • ApplyKernelByName qui prend en entrée l’image sur laquelle il faut appliquer la transformation et la transformation à appliquer, et retourne ainsi une image de type Picture. Cette méthode appelle ainsi la méthode ApplyKernel avec en paramètre l’image et le kernel associé à la transformation.
